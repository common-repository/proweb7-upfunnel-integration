== Description ==
Upfunnel integration plugin is for sending notifications to Upfunnel where you can check the marketing stats.There is no coding require to use this plugin just enable the plugin from settings and plugin will automatically doing its work.

== Installation ==
1. Donwload the plugin and extract the plugin zip file and copy and paste 'upfunnelwp' to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress and its done!!

== Frequently Asked Questions ==
 
= How plugin works =
 
When you install this plugin and you have to go to admin panel find tools section when you click the tools links you can find 'Upfunnel options' link , just click it and enable the plugin . If this field unchecked then plugin will not work.
You can get Api key from https://upfunnel.io/ using sign up.
Please note : term and condition field is required 
 
= Plugin is activated but its not working =
 
After plugin activated you need to enable the plugin from "Tools menu > Upfunnel options"
